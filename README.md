# React Boilerplate

## Prerequisites

- Node.js 18.0.0 or higher
- npm 9.0.0 or higher

## Features

- React 19
- TypeScript
- Vite
- TailwindCSS
- ESLint
- Prettier
- Radix UI
- Class Variance Authority
- Tailwind Merge
- Lucide Icons

## Getting Started

1. Clone this repository

```sh
git clone https://github.com/KelvinYou/react-boilerplate.git
```

2. Install dependencies:

```sh
npm install
```
